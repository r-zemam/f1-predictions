# 2026 F1 Season Predictions

Built for **Formula Watch** — our college F1 watch club that organizes race viewing parties and season-long engagement activities @ George Mason University.

At the start of the season, members fill out their predictions: driver championship order, constructor standings, race winners, teammate battles, storylines, and bold calls. Once submitted, the form generates a personal PDF. Members save it and don't open it until the Abu Dhabi Grand Prix finale — then we compare.

### What's on the form
- Full drivers' championship order (1–22)
- Constructors' championship order
- Teammate battles
- Iconic race winners — Silverstone, Monaco, Singapore + custom picks
- Season stats and storylines
- Driver watch — Lindblad rookie season, Hamilton at Ferrari
- Bold predictions and silly season calls

### Tech
Plain HTML, CSS, and JavaScript. No backend or login. PDF generated client-side using jsPDF. Hosted on GitHub Pages.

**Live:** https://r-zemam.github.io/f1-predictions
